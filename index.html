Projects that I'm proud of

AI for Flappy Bird

Python with pygame and python-neat

In my pursuit of teaching myself machine learning, I decided to create this project. This project uses NeuroEvolution of Augmenting Topologies (NEAT) to train AI clients to play a clone of the game Flappy Bird.

On simple tasks, the NEAT algorithm often arrives at effective networks more quickly than other contemporary neuro-evolutionary techniques and reinforcement learning modules.

This project is written in Python and uses the pygame module to recreate the Flappy bird game environment. The AI client uses the python-neat module to create neural networks that play the game. See this project on Github

Sorting Visualiser

React.js

After completing my Algorithms and Data Structures module in college, I wanted to deepen my understanding of the algorithms that we had studied. I also wanted to explore new front-end technologies that are not covered in my university, to broaden my web-development skills. Making this sorting visualiser allowed me to kill both of those birds with the one stone!

This responsive web application allow users to visualise six sorting algorithms: Quick sort, Merge sort, Tim sort, Insertion sort, Selection sort and Bubble sort. Rubik's Cube Visualiser

Processing and Java

I have been obsessed with Rubik's cubes ever since I was a child. My record is around 19 seconds for a single solve. It is not always practical to bring a Rubik's cube around with me, so I decided to make a digital one that I will always have on my laptop!

This project uses Processing, a programming language that is based on Java syntax with a focus on visualisation. This project allows users to scramble the cube and attempt to solve it themselves. If they ever get stuck, they can just press the space bar and watch the puzzle solve itself before their very eyes. See this project on Github

Microcoded Instruction Set Processor

VHSIC Hardware Description Language

This assignment was worth 70% of my overall grade in my Computer Architecture 1 module in college. I received a grade of 98% on this VHDL project. This assignment simulates a microcoded instruction set processor that can perform arithmetic, logic and shifting operations as well as conditional and unconditional branching.

The processor reads user-provided machine code from memory. All of the components that I created in this project have test-benches that ensure that their functionality is correct. See this project on Github
