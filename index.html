My name is Eoin Pinaqui and I am an undergraduate Computer Science student in Trinity College Dublin. Studying computer science has allowed me to marry my interests in technology, problem-solving and innovation. My main passions lie in software engineering and leadership.

This competition recognizes leadership potential in young people and is conducted through a series of intense interviews, covering topics such as immigration and climate change. Projects that I am proud of

Netsoc Webspaces

Collaborated on a team that created a web application that allows Dublin University Internet Society members to interact with the linux containers that their websites are hosted on. I used VueJS and Flask to build the frontend portion of the application and was heavily involved in the design of the backend, which uses Traefik and Redis to route http requests to webspaces.

Open Flow Implementation

I received 105% on my implementation of the Open Flow routing protocol with extra credit. This project allows users to send messages across a network of Open Flow switches and uses Dijkstra's Algorithm to determine the shortest path between two endpoints. This project was developed with Docker to communicate between Docker containers.

See this project on Github

Postfix and Infix Calculator

This C program allows a user to evaluate both postfix and infix expressions by passing them through the command line as paramaters. This program makes heavy use of pointer arithmetic and uses a simple stack data structure that I wrote to evaluate the results of the expressions.

See this project on Github

Connect Four

This ARM Assembly Language project allows a user to play a game of Connect 4 in the console. You can play locally with two players or a single player can take on the computer. This project uses ARM assembly language subroutines and pointer arithmetic to initialise and manipulate the connect four board. I received a grade of 98% on this project.

See this project on Github

Microprogrammed Instruction Set Processor

This implementation of a Microprogrammed Instruction Set Processor uses VHDL to simulate all of the necessary components, from multiplexers to arithmetic logic units. I completed this project as part of my Computer Architecture 1 class.

See this project on Github

TCD News

I acted as the project manager in my Programming Project 1 module, where my team and I created a social news aggregator. I created all of the widgets and queries that allow a user to navigate through over 4GB of HackerNews stories and comments. These widgets include drop-down menus, search-bars and all other navigation tools. Our team received a grade of 89% on this project.
